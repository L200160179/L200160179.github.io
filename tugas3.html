MODUL 3

Peralatan yang digunakan:
1. Arduino Uno Starter Kit
2. 1 buah resistor 220 ohm
3. 1 buah 7-segment
4. Potensiometer

Untuk menyelesaikan tugas modul ini, kita pasang dan sesuaikan rangkaian di breadboard menurut dasar yang diajarkan di modul.

TUGAS I
Membuat 7-Segment menampilkan angka dari 0 hingga 9 dengan delay 1 detik.


Source Code: 
		int val[]={B11000000,B11111001,B10100100,B10110000,B10011001,B10010010,B10000010,B11111000,B10000000,B10010000};
void setup() {
  // put your setup code here, to run once:
  DDRD = B11111111;
}

void loop() {
  // put your main code here, to run repeatedly:
  for(int i=0; i<10; i++){
    PORTD = val[i];
    delay(1000);
    }
}

TUGAS II
Membuat 7-Segment menampilkan angka dari 0 hingga 9 dengan potensio sebagai controlnya.


Source Code: 

			int sensorPin = A0;
int sensorValue = 0;
int val[]={B11000000,B11111001,B10100100,B10110000,B10011001,B10010010,B10000010,B11111000,B10000000,B10010000};
void setup() {
  // put your setup code here, to run once:
  DDRD = B11111111;
}

void loop() {
  // put your main code here, to run repeatedly:
  sensorValue = analogRead(sensorPin);
  if(sensorValue > 900){
    PORTD = val[9];
    }
    else if (sensorValue > 800){
      PORTD = val[8];
      }
      else if (sensorValue > 700){
        PORTD = val[7]; 
        }
        else if (sensorValue > 600){
          PORTD = val[6]; 
          }
          else if (sensorValue > 500){
            PORTD = val[5]; 
            }
            else if (sensorValue > 400){
              PORTD = val[4];
              }
              else if (sensorValue > 300){
                PORTD = val[3];
                }
                else if (sensorValue > 200){
                  PORTD = val[2];
                  }
                  else if (sensorValue > 100){
                    PORTD = val[1];
                    }
                    else if (sensorValue > 0){
                      PORTD = val[0];
                      }
}
