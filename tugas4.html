Peralatan yang digunakan:
1. Arduino Uno Starter Kit
2. 1 buah resistor 1KOhm dan 220 Ohm
3. 1 buah LED 4. 1 buah 7-segment

Untuk menyelesaikan tugas modul ini, kita pasang dan sesuaikan rangkaian di breadboard menurut dasar yang diajarkan di modul.

TUGAS I
Menggunakan LDR sebagai sensor cahaya untuk mengontrol tingkat kecerahan LED.


Source Code: 
		int potPin = A0;

  // put your setup code here, to run once:
 
}

  potValue = analogRead(potPin);

}
